# Juego de Carros 🚗🏁  

Un sencillo juego de carreras hecho con HTML, JavaScript y PHP, que almacena el nombre del usuario en una base de datos MySQL (MariaDB).  

## 📂 Estructura del proyecto  
- `index.html` → Página principal  
- `juego.html` → Página del juego  
- `registrar.php` → Guarda jugadores en la base de datos  
- `ver_jugadores.php` → Muestra los jugadores registrados  
- `config.js` → Configuración del juego
